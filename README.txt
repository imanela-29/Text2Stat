Installation:
python déjà installé
creer un dossier global
dans cmd executer les commandes:
python -m venv my_env  (environnement virtuelle sera crée, un dossier my_env sera crée dans dossier global)
my_env\Scripts\activate (pour activer my_env)
pip install streamlit
pip install awesome-streamlit
pip install requests
pip install beautifulsoup4
pip install regex
pip install pyaramorph
pip install wordcloud
pip install lxml
pip install nltk
pip install -U spacy
pip install stop-words
pip install python-bidi
pip install arabic_reshaper
pip install matplolib
copier/coller dossier 'app' dans dossier global
sur même cmd executer :
cd app
streamlit run app.py  (pour executer l'application, s'ouvre dans navigateur)
